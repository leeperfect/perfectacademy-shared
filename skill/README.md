# 通用技能目录

## 目录结构

- **teaching_skills/**：通用授课技巧
  - 课堂管理技巧
  - 语言表达技巧
  - 互动教学策略
  - 多媒体教学方法

- **training_methods/**：通用培训方法
  - 培训课程设计
  - 学员评估体系
  - 个性化学习方案
  - 培训效果评估

- **case_analysis/**：通用案例分析
  - 教学案例分析
  - 培训效果案例
  - 学员成长案例
  - 行业最佳实践

- **experience/**：通用教学经验
  - 教学管理经验
  - 培训师成长路径
  - 行业发展趋势
  - 教学创新探索

## 内容规范

- **文件命名**：`[技能类型]_[具体内容]_[版本号]_[日期].md`
  - 示例：`teaching_skills_课堂互动技巧_v1.0_20260406.md`

- **内容结构**：
  1. 技能概述
  2. 具体方法
  3. 应用场景
  4. 案例说明
  5. 注意事项

## 版本管理

- 使用语义化版本号：`v[主版本].[次版本].[修订号]`
- 每次更新需在CHANGELOG.md中记录变更内容
- 定期回顾和更新技能内容，保持时效性
